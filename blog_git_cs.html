Git Cheat Sheet

$ git init

initializing new local repo

$ git status

shows modified files in local repo

$ git log

shows previous commits

$ git add "file": example: git add robot.java

puts file on stage

$ git add .

stages all files in the path

$ git commit -m "massage"

commits all staged files

$ git checkout "hash"

change to previous commits

$ git checkout "spec. branch"

change to spec. branch

$ git branch

check out branches

$ git branch "branch name"

create new branch

$ git branch -d "newbranch"

remove branch

$ git merge "Branch_that_should_be_merged"

Merges current branch with Branch_that_should_be_merged

$ git remote add 'remote_repo_name' 'your_url_name'

add remote repo

$ git remote remove origin

remove remote repo

$ git remote -v

shows remote repos

$ git push -u 'remote_repo' 'branch'

pushes local repo to remote repo

$ git pull 'remote__repo_name' 'branch_name'

pulls remote repo to local

$ git pull 'remote__repo_name' 'branch_name' --allow-unrelated-histories

pulls remote repo to local --BY FORCE

$ git clone 'HTTPS_Link_repo'

clones remote repo to current dir

$ git pull

after clone -> pull request
updates current cloning

$ git checkout --track origin/BRANCH'

Pull remote branch to local machine
Branch doesnt have to be created locally
depending on git version, slight differences possible
lookup: https://stackoverflow.com/questions/9537392/git-fetch-remote-branch

$ git log --graph --pretty=oneline --abbrev-commit

Display git log in graph like fashion
